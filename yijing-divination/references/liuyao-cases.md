# 六爻实战案例集

> 来源：《增删卜易》《卜筮正宗》经典案例
> 提取目的：提供实战案例，帮助理解六爻断卦的实际应用
> 整理时间：2026年6月12日

---

## 案例格式说明

每个案例包含以下要素：
- **问题**：占问何事
- **起卦**：本卦、变卦、动爻
- **用神**：取何爻为用神
- **断卦**：分析过程（用神旺衰、原忌仇神、旬空月破等）
- **结果**：实际应验情况
- **启示**：从此案例中学到的断卦要点

## 一、求财类

### 案例1：占求财（兑为泽变雷水解）

**问题**：酉月辛亥日，占求财。

**起卦**：
- 本卦：兑为泽（第58卦）
- 变卦：雷水解（第40卦）
- 动爻：九二（兄弟酉金）、九五（官鬼巳火）

**卦象**：
```
六爻：丶丶父母未土（世）
五爻：○兄弟酉金 → 兄弟申金（动）
四爻：丶子孙亥水
三爻：丶父母丑土（应）
二爻：丶妻财卯木
初爻：○官鬼巳火 → 妻财寅木（动）
```

**用神**：妻财爻（卯木）

**断卦**：
1. 用神分析：卯木财爻为用神，酉月占卦，金旺木衰，用神月破
2. 原神分析：巳火官鬼动化寅木财爻，原神动而化财
3. 忌神分析：酉金兄弟动，忌神动而克用
4. 综合判断：
   - 卯木财爻空而且破，被酉金忌神克制
   - 但巳火原神动化寅木财爻，"神兆机于动"
   - 巳火化空，目下不见，待甲寅日出空而相见
   - 寅木之财以生官，官来生世

**结果**：果于甲寅日早，见木日得财。

**启示**：
- 用神虽空破，但原神动化财爻，"神兆机于动"
- 化空之爻，待出空之日应验
- 原神动化用神，接续相生，吉

### 案例2：占求财（火天大有）

**问题**：寅月庚戌日，占求财。

**起卦**：
- 本卦：火天大有（第14卦）
- 变卦：无（静卦）
- 动爻：无

**卦象**：
```
六爻：丶官鬼巳火（应）
五爻：丶丶父母未土
四爻：丶兄弟酉金
三爻：丶父母辰土（世）
二爻：丶妻财寅木
初爻：丶子孙子水
```

**用神**：妻财爻（寅木）

**断卦**：
1. 用神分析：寅木财爻为用神，寅月占卦，用神值月建，旺相
2. 旬空分析：寅木财爻旬空（甲辰旬，寅卯空）
3. 综合判断：
   - 用神值月建，旺相有力
   - 但旬空，"目下尚空，要到甲寅日出空可得"
   - "逢空亦空，终非落底之空"
   - 值月建乃为旺，待出空之日则不空

**结果**：甲寅日出空，得财。

**启示**：
- 用神值月建，虽旬空不为真空
- "逢空亦空，终非落底之空，逢伤亦伤却有待时之用"
- 待出空之日应验

## 二、疾病类

### 案例3：自占病（泽风大过变火风鼎）

**问题**：巳月乙未日，自占病。

**起卦**：
- 本卦：泽风大过（第28卦）
- 变卦：火风鼎（第50卦）
- 动爻：六三（妻财丑土）、上六（妻财未土）

**卦象**：
```
六爻：×妻财未土 → 子孙巳火（动）
五爻：丶官鬼酉金
四爻：丶妻财未土（世）
三爻：丶官鬼酉金
二爻：丶父母亥水
初爻：×妻财丑土（应）→ 父母子水（动）
```

**用神**：世爻（亥水）

**断卦**：
1. 用神分析：自占病，世爻亥水为用神
2. 忌神分析：未土忌神动而克水
3. 原神分析：酉金元神亦动，忌神未土反生元神之酉金，金生亥水，接续相生
4. 综合判断：
   - 用神亥水月冲日克，值月破而被克
   - 虽有生扶之不起，"如树无根，寒谷不回春也"
   - 用神无根，元神有力亦难生

**结果**：果卒于癸卯日。应卯日者，冲去元神之日也。

**启示**：
- 用神无根，元神有力亦难生
- 月破之爻，虽有生扶不起
- "如树无根，寒谷不回春"

### 案例4：占父母病

**问题**：占父母病。

**用神**：父母爻

**断卦要点**：
- 父母爻旺相，或临日月动爻相生 → 病愈
- 父母爻衰弱，或被日月动爻冲克 → 病重
- 近病逢冲逢空，不药而愈
- 久病逢冲逢空，灵丹莫救

**启示**：
- 占父母病以父母爻为用神
- 近病与久病的断法不同
- 用神旺衰是关键

## 三、功名类

### 案例5：占功名

**问题**：占功名。

**用神**：官鬼爻

**断卦要点**：
- 旺官持世，或日月动爻作官星生合世爻 → 求名如拾芥
- 子孙持世 → 失望
- 官星旺相，无冲无克 → 功名可成

**启示**：
- 占功名以官鬼爻为用神
- 官星持世或生世为吉
- 子孙持世为凶（子孙克官）

## 四、婚姻类

### 案例6：占婚姻

**问题**：占婚姻。

**用神**：妻财爻（男占）、官鬼爻（女占）

**断卦要点**：
- 用神旺相，生合世爻 → 婚姻可成
- 用神衰弱，冲克世爻 → 婚姻难成
- 世应相生相合 → 宾主相投
- 世应相克相冲 → 两情不睦

**启示**：
- 占婚姻看世应关系
- 用神旺衰是关键
- 世应生克定吉凶

## 五、出行类

### 案例7：占出行

**问题**：占出行。

**用神**：世爻

**断卦要点**：
- 世爻旺相，或日月动爻生合世爻 → 出行顺利
- 世爻衰弱，或被日月动爻冲克 → 出行不利
- 世爻旬空 → 心中无底，出行不果
- 世爻月破 → 出行必有灾咎

**启示**：
- 占出行以世爻为用神
- 世爻旺衰定吉凶
- 旬空月破为凶兆

## 六、失物类

### 案例8：占失物

**问题**：占失物。

**用神**：妻财爻

**断卦要点**：
- 用神旺相，生合世爻 → 失物可寻
- 用神衰弱，冲克世爻 → 失物难寻

## 八、综合断卦要点

### 1. 用神取用速查

| 所问之事 | 用神 |
|---------|------|
| 求财 | 妻财爻 |
| 功名 | 官鬼爻 |
| 疾病 | 世爻（自占）、六亲爻（代占） |
| 婚姻 | 妻财爻（男）、官鬼爻（女） |
| 出行 | 世爻 |
| 失物 | 妻财爻 |
| 官司 | 官鬼爻 |

### 2. 断卦核心逻辑

**用神旺衰**：
- 用神旺相，或临日月动爻生扶 → 吉
- 用神衰弱，或被日月动爻冲克 → 凶

**原忌仇神**：
- 原神动而生用 → 吉
- 忌神动而克用 → 凶
- 仇神动而克原生忌 → 凶

**旬空月破**：
- 用神旬空 → 事不落实，待出空应验
- 用神月破 → 事必不成，待出月或填实

**进退神**：
- 用神化进神 → 事越来越好
- 用神化退神 → 事越来越差

### 3. 应期判断

**出空**：旬空地支在日辰上出现之日
**填实**：旬空地支在月建上出现
**冲空**：旬空地支被日辰冲
**合空**：旬空地支被日辰合
