# 奇门遁甲 · 实战断法总纲

> 来源：奇门法窍、御定奇门宝鉴、奇门遁甲元灵经

---

## 一、核心判断原则

### 1. 急则从神，缓则从门
- **事情紧急**：以八神为优先判断标准（神煞速应）
- **事情不急**：以八门为优先判断标准（门主长远）

### 2. 主客之分
| 角色 | 谁动 | 看什么 |
|------|------|--------|
| 主动去找人/出门 | 我动 | 时干宫 + 天盘星 |
| 等人来找/在家 | 人动 | 日干宫 + 地盘干 |

### 3. 三盘权重
| 盘 | 权重 | 主事 |
|----|------|------|
| 天盘（九星） | 大环境、天时 | 不可控因素 |
| 中盘（八门） | 人事、人和 | 人际关系、行动方向 |
| 地盘（方位） | 地利、根基 | 基础条件、稳定性 |

## 二、用神取用法

### 按人物取用神
| 人物 | 用神 |
|------|------|
| 自己/问事人 | 日干落宫 |
| 对方/他人 | 时干落宫（或应事之日干） |
| 上级/领导 | 值符落宫 |
| 长辈/父母 | 年干落宫 |
| 同辈/朋友 | 月干落宫 |

### 按事项取用
| 事项 | 用神 |
|------|------|
| 求财 | 生门落宫 + 戊土 |
| 求官/求职 | 开门落宫 + 值符 |
| 婚姻/感情 | 乙（女方）庚（男方）落宫 |
| 疾病/健康 | 天芮星落宫，天心星制病 |
| 出行/远行 | 时干宫 + 八门吉凶 |
| 失物 | 时干宫 + 玄武 |
| 考试/文书 | 丁奇 + 景门 |
| 官司/诉讼 | 惊门 + 白虎 + 值符 |
| 谈判/交易 | 开门 + 六合 |

## 三、宫位生克判断

| 关系 | 吉凶 | 含义 |
|------|------|------|
| 用神宫生日干宫 | 大吉 | 事来就我，得利 |
| 日干宫生用神宫 | 泄气 | 我去就事，费力 |
| 用神宫克日干宫 | 大凶 | 事来克我，不可为 |
| 日干宫克用神宫 | 可成 | 我可制事，但需努力 |
| 用神宫与日干宫比和 | 吉 | 顺畅无阻 |

## 五、吉方选择三步法

1. **看八门**：避开死惊伤三门，优先选择开休生三门所在宫
2. **看星神**：吉星吉神加临更佳，凶星凶神减分
3. **看格局**：有无特殊吉格（青龙返首等）或凶格（青龙逃走等）
4. **综合判断**：三门皆吉 → 大吉；二吉一凶 → 可用；一吉二凶 → 不宜
