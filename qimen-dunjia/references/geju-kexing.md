## 四、按问事速查

| 问事 | 看什么 |
|------|--------|
| 出行 | 时干宫的门和星是否吉利。开门/休门/生门为吉，死门/惊门不宜行 |
| 求财 | 看生门宫。生门临戊+丙（青龙返首）大发，临白虎猖狂有损 |
| 求官/求职 | 看开门宫。开门临值符有贵人，临螣蛇有机变 |
| 感情婚姻 | 看乙庚落宫。乙（女方）庚（男方）两宫相生则吉，相克则难 |
| 疾病 | 看天芮星、病符、死门。天心+乙奇为良医 |
| 失物 | 看时干宫。生门临玄武→被人偷，找到的几率看宫位生克 |
| 考试 | 看丁奇、景门。丁+景门吉利则文采过人 |
| 官司 | 看惊门、白虎。白虎猖狂大凶，值符临开门则得贵人解围 |

## 五、找吉方速查

| 目的 | 找什么 |
|------|--------|
| 求财吉利方 | 生门所落之宫的方向 |
| 求见贵人方 | 开门所落之宫的方向 |
| 隐蔽躲藏方 | 杜门所落之宫的方向 |
| 出行有利方 | 开门+休门+生门任一落宫方向 |
| 必须避开的方 | 死门+惊门+伤门落宫方向 |
| 谈判有利方 | 值符所落之宫方向 |
