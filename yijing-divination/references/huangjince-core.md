# 黄金策 · 总断千金赋核心规则

> 来源：《黄金策》题明·刘基著。本篇为全书纲领，六爻断卦的核心理论基石。

## 一、日辰主宰

**日辰是六爻断卦的最重要参照。** 不看日辰则不知卦中吉凶轻重。

| 日辰能力 | 说明 |
|---------|------|
| 冲动静爻 | 安静之爻被日辰冲则为暗动，以动爻论 |
| 制动爻 | 动爻被日辰克制，减缓其力 |
| 抑凶神 | 克制凶神使之无害 |
| 挫旺相 | 即使旺相，日辰可挫其锐 |
| 扶衰弱 | 衰弱之爻得日辰生扶则有用 |
| 破生合 | 日辰能破坏卦爻之间的合 |
| 合成忌神 | 日辰也能合成忌神来害用神 |

**关键原则**：日辰生扶用神则吉，生扶忌神则凶；克制忌神则吉，克制用神则凶。

## 二、月建提纲

**月建与日辰一样能生扶拱合、冲克刑害卦爻。**

| 月建作用 | 说明 |
|---------|------|
| 定旺相休囚 | 同我者旺、我生者相、生我者休、克我者囚、我克者死 |
| 月破 | 与月建相冲的地支为月破，月破之爻当月无力 |
| 缺用神时 | 卦中无用神，月建是则待当月可得 |
| 克用神时 | 月建克用神，须过此月才能成功 |

**月建与日辰的分别**：月建定旺相休囚和月破，日辰定生旺墓绝空。

---

## 三、生扶拱合 vs 克害刑冲

**四吉四凶总纲：**

| 类型 | 包含 | 作用 | 对用神 | 对忌神 |
|------|------|------|--------|--------|
| 四吉 | 生、扶、拱、合 | 调护帮助 | 弱变强、衰变旺、伏得起 | 凶 |
| 四凶 | 克、害、刑、冲 | 伤害克制 | 凶 | 吉 |

- **生**：五行相生（金生水、水生木等）
- **扶**：同类相帮，先帮后为扶
- **拱**：同类相帮，后帮先为拱
- **合**：二合（子丑合）、三合（申子辰合水局）
- **克**：五行相克（金克木等）
- **害**：六害（子未害、丑午害等）
- **刑**：三刑、四刑
- **冲**：六冲（子午冲等）

## 四、太过与不及

**卦以中和为好，不要太过，不要不及。**

| 情况 | 处理 |
|------|------|
| 太过（用神太多太旺） | 损之：用月建、日辰、动爻来克制 |
| 不及（用神衰弱） | 益之：用生、扶、拱、合来增强 |

---

## 五、旺相休囚

**依月建定：**

| 状态 | 与月建关系 | 含义 |
|------|-----------|------|
| 旺 | 五行相同 | 最强，占事有力 |
| 相 | 我生之 | 次强，有气 |
| 休 | 生我之 | 无力 |
| 囚 | 克我之 | 衰弱 |
| 死 | 我克之 | 最弱 |

## 六、入墓

**入墓之爻不能克他爻，也不受他爻所克。**

| 五行 | 墓库 |
|------|------|
| 木 | 未 |
| 火 | 戌 |
| 金 | 丑 |
| 水、土 | 辰 |

**墓中人，不冲不发**：用神入墓则阻滞费力，须待日辰、动爻冲克其墓日，方可有为。

---

## 七、暗动

**静爻被日辰所冲，即为暗动。**

暗动以动爻论——静爻原本不发挥力量，被日辰冲动后突然产生作用。这是断卦中容易忽略的变数。

---

## 八、动爻逢冲

| 爻的状态 | 逢冲结果 |
|---------|---------|
| 旬空安静之爻 | 逢冲则起，有用了 |
| 旬空发动之爻 | 逢冲为实，脱空而出 |
| 安静不空之爻 | 逢冲为暗动，以动爻论 |
| **发动不空之爻** | **逢冲为散、为脱，事散脱不成** |

---

## 九、贪生贪合忘克忘刑

**原本能刑冲克害他爻的爻，遇到合就忘了害。**

贪合忘刑、贪合忘冲、贪合忘克、贪生忘克——四者一理。爻遇到更优先的关系（生、合），原有的克害刑冲关系就退让。

## 十、空亡的处理

| 空亡类型 | 规则 |
|---------|------|
| 无故而空 | 无刑冲克害而身世/主象自落空亡 → 大凶 |
| 有故而空 | 因日月动爻伤克而避入空亡 → 避祸，过旬可用 |
| 空逢冲 | 空爻被冲则动，动则不空，有用 |
| 带旺匪空 | 旺相之爻值旬空不以空论，过旬即成 |
| 空化空 | 动爻空化出之爻又空，非真空，出旬有用 |

## 十一、世应关系

| 规则 | 说明 |
|------|------|
| 世为己，应为人 | 世代表自己，应代表他人 |
| 世应生合比和 | 谋事有成 |
| 世应刑冲克害 | 必主艰难 |
| 世应俱空 | 彼此皆不实，谋事有阻 |
| 世动 | 自己三心二意 |
| 应动 | 他人心意变迁 |

---

## 十二、进神与退神

爻动变出同类五行时为「动变比和」，需区分进退：

| 方向 | 示例 | 含义 |
|------|------|------|
| 进神 | 寅木化卯木 | 力量倍增，吉凶势头加剧 |
| 退神 | 卯木化寅木 | 力量渐减，吉凶势头消退 |

---

## 十三、绝处逢生

用神处于绝地，若得日辰、动爻来生，为绝处逢生。看似绝望，实有转机。

---

## 十四、期日判断速查

| 用神状态 | 断期方法 |
|---------|---------|
| 旺相不动 | 冲动之月日 |
| 有气发动 | 合日或本日 |
| 受制 | 制杀之月日 |
| 太旺（旺动+生扶） | 入墓之月日 |
| 无气发动遇生扶 | 生扶之月日 |
| 入墓 | 破墓之月日 |
| 空亡 | 冲动/过旬之月日 |
| 合住 | 冲破合之月日 |
| 休囚 | 生旺之月日 |
| 散事 | 死墓绝之月日 |
