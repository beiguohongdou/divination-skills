# 大六壬 · 天地盘排法

> 天地盘是大六壬起课的底层结构。地盘固定不变，天盘随占时月将流转。
> 来源：六壬学基本认识、六壬大全、大六壬心镜

---

## 一、地盘（固定不变）

十二地支按子丑寅卯辰巳午未申酉戌亥顺序排列，子居下、午居上、卯居左、酉居右：

```
        巳  午  未  申
      辰            酉
      卯            戌
        寅  丑  子  亥
```

地盘上的方位：子为北、午为南、卯为东、酉为西。

月将以**中气**为界。每月一个中气，太阳在此时进入下一个黄道宫位：

## 三、天盘（月将加时）

**口诀：月将加占时，顺行十二辰。**

将当月月将的地支放到占时（时辰地支）的位置上，然后按地支顺序（子丑寅卯辰巳午未申酉戌亥）顺时针排布十二个月将。

**示例**：正月雨水后，月将为亥（登明）。若占时为午时，则将亥放到地盘午的位置上，然后顺排：亥（午）、子（未）、丑（申）、寅（酉）、卯（戌）、辰（亥）、巳（子）、午（丑）、未（寅）、申（卯）、酉（辰）、戌（巳）。

```
天盘在各地盘位置的对应：
地盘：子  丑  寅  卯  辰  巳  午  未  申  酉  戌  亥
天盘：巳  午  未  申  酉  戌  亥  子  丑  寅  卯  辰
```

## 四、占时

**占时是起课的时间基点。** 通常取用户问事的时辰（按地支时辰）。

十二时辰对应关系：

| 时辰 | 时间 | 时辰 | 时间 |
|------|------|------|------|
| 子 | 23:00-01:00 | 午 | 11:00-13:00 |
| 丑 | 01:00-03:00 | 未 | 13:00-15:00 |
| 寅 | 03:00-05:00 | 申 | 15:00-17:00 |
| 卯 | 05:00-07:00 | 酉 | 17:00-19:00 |
| 辰 | 07:00-09:00 | 戌 | 19:00-21:00 |
| 巳 | 09:00-11:00 | 亥 | 21:00-23:00 |
