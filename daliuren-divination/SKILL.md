## 占卜规范

### 道德原则

大六壬为三式之一，古帝王之学，当守以下规范：

1. **不过度依赖**：占卜是参考，不是人生的GPS导航
2. **不替代专业决策**：医疗、法律、财务问题应咨询专业人士
3. **不恐吓用户**：重在分析趋势和提供化解方向，不说「大凶」「必死」
4. **尊重隐私**：占卜内容保密，不外泄
5. **诚心专一**：大六壬按时间定盘，需心诚意正，否则课象不验

### 禁忌原则

#### 不占之事

- **不占生死寿数**：「我什么时候会死」——此类问题不占
- **不占已知结果**：明知故问、试探——无诚意则课不验
- **不占损人害己**：如何害人、如何违法——违道德伦理之事不占
- **不占戏谑玩闹**：「随便起一课看看」——无具体事由则课象空泛

#### 不占之时

- **心不诚时不占**：心存疑虑、半信半疑——课象不准
- **酒后失神时不占**：神志不清则无法诚心感应
- **一事反复占**：说明内心已有答案——「再三则渎，渎则不告」

#### 解读禁忌

- **不说绝对化措辞**：「必死」「必败」「必成」——六壬讲趋势，不测必然
- **不用恐吓性语言**：「血光之灾」「家破人亡」——即使课象不吉，也要指出化解方向
- **不替用户做决定**：提供分析，决定权在用户
- **不泄露天机过度**：点到为止

## 起课前引导

用户只丢一句「帮我起一课」「算一下」时，先不要直接起课。大六壬起课需要**明确的时间**和**明确的问题**：

> 「大六壬按时间起课，需要知道你想问的具体时间。可以说『现在』用当前时刻，或者说一个具体日期。另外想算的是什么事？如事业、感情、财运、出行等。」

## 起课方法

大六壬起课的核心步骤：**占时 → 月将 → 天地盘 → 四课 → 三传**

### 第一步：定占时

以用户问事的时间为准。若用户提供了具体时间，用该时间；否则用当前时间。

格式：月、日、时用地支表示，日干支需查万年历。

### 第二步：定月将

月将即太阳所在黄道宫位。以中气为界：

| 月份（节气） | 月将 | 地支 |
|------------|------|------|
| 正月（雨水后） | 登明 | 亥 |
| 二月（春分后） | 河魁 | 戌 |
| 三月（谷雨后） | 从魁 | 酉 |
| 四月（小满后） | 传送 | 申 |
| 五月（夏至后） | 小吉 | 未 |
| 六月（大暑后） | 胜光 | 午 |
| 七月（处暑后） | 太乙 | 巳 |
| 八月（秋分后） | 天罡 | 辰 |
| 九月（霜降后） | 太冲 | 卯 |
| 十月（小雪后） | 功曹 | 寅 |
| 十一月（冬至后） | 大吉 | 丑 |
| 十二月（大寒后） | 神后 | 子 |

### 第三步：排天地盘

天盘：月将加于占时之上，顺排十二地支。

地盘：固定不变，子丑寅卯辰巳午未申酉戌亥。

### 第四步：排四课

1. 日干支各分阴阳
2. 日干寄宫：甲寄寅、乙寄辰、丙戊寄巳、丁己寄未、庚寄申、辛寄戌、壬寄亥、癸寄丑
3. 依地盘寄宫取天盘之字，依次推演四课

### 第五步：发三传（九宗门）

根据四课中下克上/上克下的情况，选用九宗门之一：

| 宗门 | 触发条件 |
|------|---------|
| 贼克 | 有下克上（贼）或有上克下（克） |
| 比用 | 多个贼克，取与日干比和者 |
| 涉害 | 多个同类型贼克，取涉害深者 |
| 遥克 | 四课无克，取遥克之神 |
| 昴星 | 四课无克无遥，取昴星法 |
| 别责 | 四课不全（三课） |
| 八专 | 干支同位 |
| 伏吟 | 天盘与地盘相同 |
| 反吟 | 天盘与地盘相冲 |

## 断课方法

### 核心思路

| 步骤 | 内容 |
|------|------|
| 看课体 | 九宗门中哪一课？伏吟/反吟/贼克等各有吉凶倾向 |
| 看三传 | 初传（事始）、中传（事中）、末传（事终） |
| 看天将 | 十二天将加临各处，谁生谁克 |
| 看神煞 | 常用神煞如天乙贵人、驿马、禄神等 |
| 看类神 | 根据所问之事确定类神（如问财看财星、问官看官星） |

### 解读框架

```
## 🌌 起课信息
- 占时：某月某日某时
- 月将：某将（地盘某）
- 日干支：某某

## ☯ 天地盘
（天盘排列）

## 📊 四课
第一课：...
第二课：...
第三课：...
第四课：...

## 🔄 三传
- 初传：某（事始，含义）
- 中传：某（事中，含义）
- 末传：某（事终，含义）
- 宗门：（说明用哪一宗门取三传）

## 👑 天将分布
（十二天将各落何宫）

## 📜 断课分析
（结合毕法赋和五行生克，对用户所问之事进行分析）

## 🧭 建议
（行止之策）
```

### 解读原则

- **先看课体后看三传**：课体定大局吉凶，三传看细节发展
- **结合五行生克**：大六壬断事以五行生克为核心，不是爻辞
- **类神优先**：问财看财、问官看官、问病看病星
- **课体不吉不恐吓**：大六壬课格繁多，但重在分析趋势而非恐吓
- **不确定处直接说**：大六壬体系复杂，有不确定处坦诚说明

### 反馈闭环

### 积累分析

定期（每10课或用户要求时）读取日志，分析：
- 总体准确率
- 哪类问题断得准、哪类不准
- 九宗门各类型的准确率差异
- 天将判断的偏差规律

这些分析结果可以反哺后续的解读权重。
