## 一、野鹤简易断卦法（零基础可用）

野鹤为完全不懂五行生克的人设计了一套最简断法：

> 学会点卦 → 在八宫全图内寻出此卦 → 照样装排世应五行六亲 → 单看世爻

**四大类事断法**：

| 所问之事 | 世爻断法 |
|---------|---------|
| 防忧虑患 | 子孙持世 → 无忧；官鬼持世 → 忧疑难解 |
| 求功名 | 官鬼持世 → 可成；子孙持世 → 待时 |
| 求财 | 妻财持世 → 必得；兄弟持世 → 难求 |
| 疾病 | 近病逢六冲卦 → 不药而愈；久病逢六冲 → 难调 |

**前提**：此简易法仅适用于完全不懂五行生克之人。稍懂五行者，神会示现隐微之卦，不可用此法。

## 二、四处生克冲合（核心框架）

野鹤提出，用神会受到**四个来源**的生克冲合：

| 来源 | 说明 | 示例 |
|------|------|------|
| **月建** | 月令五行能生克冲合用神 | 戌土用神在寅卯月被克，在巳午月得生 |
| **日辰** | 日辰五行能生克冲合用神 | 辰日冲动戌土为暗动，巳午日生戌土为旺 |
| **动爻** | 卦中动爻能生克冲合用神 | 寅木动克戌土，午火动生戌土 |
| **自变** | 用神自发动变出的爻回头生克 | 变出巳午火为回头生，变寅卯木为回头克 |

**综合判断法则**：

| 四处生克比例 | 断法 |
|-------------|------|
| 全来生合 | 诸占全吉 |
| 三生一克 | 以吉断 |
| 两生两克 | 看旺衰——生用之神旺相则吉，克用之神旺相则凶 |
| 一生三克 | 生用之神旺相→绝处逢生/凶中得解；生用之神休囚→有名无实/同大凶 |
| 全来克 | 诸占大凶 |

## 三、多占之法（野鹤独创）

**「一事多占」是野鹤老人最核心的实战方法论。**

> 事之缓者，迟迟再占；事之急者，歇歇又占。不拘早晚，不必焚香。深更半夜亦可占之。只要单为此事而占，不可又占他事。

**规则**：

| 情况 | 做法 |
|------|------|
| 卦中世爻/用神不现 | 再占一卦，再不见明日又占 |
| 已得明确指示（如官鬼持世/兄弟持世） | 不必再占，不可强求改卦 |
| 一事涉及多人 | 各占一卦，有一卦得吉则皆吉 |
| 心怀两三事而占 | 非一念之诚，决无灵验 |

## 四、世爻为核心

野鹤强调：**自占吉凶以世爻为用神，不以卦名论吉凶。**

占不同对象用不同用神：

| 占问对象 | 用神 |
|---------|------|
| 自己吉凶 | 世爻 |
| 父母/文书/房屋 | 父母爻 |
| 兄弟/姐妹 | 兄弟爻 |
| 妻妾/财物 | 妻财爻 |
| 子孙/解忧 | 子孙爻 |
| 功名/官非/鬼祟 | 官鬼爻 |
| 他人之事 | 应爻 |

## 五、世爻临不同六亲的快速判断

| 世临 | 含义 | 吉凶 |
|------|------|------|
| 子孙持世 | 福德临身 | 防灾无患、解忧消灾 |
| 官鬼持世 | 忧患临身 | 防忧虑患则凶、求功名则吉 |
| 妻财持世 | 财福临身 | 求财可得 |
| 兄弟持世 | 劫财之神 | 求财难、合伙防破财 |
| 父母持世 | 文书临身 | 占文书可得、但克子孙 |

## 六、六冲卦快速判断

**八宫之首卦皆为六冲卦**：乾为天、坎为水、艮为山、震为雷、巽为风、离为火、坤为地、兑为泽。

另加：天雷无妄、雷天大壮，共十卦。

| 疾病类型 | 六冲卦含义 |
|---------|-----------|
| 近病 | 逢六冲不药而愈 |
| 久病 | 逢六冲妙药难调 |
