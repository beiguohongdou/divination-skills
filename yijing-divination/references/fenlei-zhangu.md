# 分类占断指南

本参考综合梅花易数体用生克法与六爻纳甲用神法，提供常见占问类别的断法要点。占断时先定体用（梅花易数）或用神（六爻纳甲），再结合万物类象和爻辞解读。

---

## 一、天时占（问天气晴雨）

**梅花易数法**：看卦象推断天气变化。
- 离火为体 → 宜晴；坎水为体 → 宜雨
- 乾兑金 → 明爽；震巽木 → 有风；坤艮土 → 阴云
- 互卦见离 → 转晴；互卦见坎 → 转雨
- 坎兑相遇 → 有雨雪；离震相遇 → 有雷电

**六爻法**：以父母爻为雨雪，子孙爻为晴天，妻财爻为晴明，官鬼爻为雷电浓雾，兄弟爻为风云。

---

## 二、婚姻占

**梅花易数法**：
- 体卦旺相 → 自身条件好；用卦生体 → 对方主动或对方条件有利于己
- 体用比和 → 两情相悦，婚姻可成
- 用克体 → 对方不满意或有阻力；体生用 → 自己付出多
- 兑为少女、震为长男、离为中女、坎为中男 → 结合看年龄匹配
- 互卦见兑乾 → 婚介或长辈牵线；见坎 → 有波折

**六爻法**：
- 男占婚姻 → 以妻财爻为用神，应爻为对方家庭
- 女占婚姻 → 以官鬼爻为用神（代表丈夫），应爻为对方家庭
- 世应相生相合 → 婚姻可成；世应相冲相克 → 难成或婚后不睦

---

## 三、求财占

**梅花易数法**：
- 体克用 → 有财，需主动争取
- 用生体 → 有财，有人送上门
- 体生用 → 耗财，付出多于收获
- 用克体 → 无财，甚至破财
- 乾兑金为用 → 金属、珠宝、金融之财
- 坤艮土为用 → 地产、农业、土产之财
- 离火为用 → 文书、文化、炉冶之财
- 震巽木为用 → 竹木、茶叶、山林之财
- 坎水为用 → 水产、酒货、运输之财

**六爻法**：
- 以妻财爻为用神
- 子孙爻为元神（生财，代表财源）→ 子孙旺相则有财路
- 兄弟爻为忌神（克财，代表破财竞争）→ 兄弟旺动则破财
- 官鬼爻 → 耗财，但适度的官鬼代表通过权力/职位得财

---

## 四、求名占（功名、考试、晋升）

**梅花易数法**：
- 体克用 → 可成，需靠自己实力
- 用生体 → 顺利，有贵人相助
- 乾兑金为体 → 利武职、刑官、权力之位
- 离火为体 → 利文职、文书之官
- 震巽木为体 → 利号令、风宪之职

**六爻法**：
- 以官鬼爻为用神（代表功名、官职）
- 父母爻为元神（生官鬼，代表文书、考试成绩）
- 子孙爻为忌神（克官鬼）→ 子孙持世则求名难成
- 妻财爻 → 生官鬼，有助力

---

## 五、疾病占

**梅花易数法**：
- 体卦旺相 → 病情轻、易愈
- 用克体 → 病情重
- 体生用 → 康复慢、消耗大
- 用生体 → 得良医好药，易康复
- 乾为头面、肺、筋骨；坤为腹、脾、胃；震为足、肝；巽为股、气
- 坎为耳、肾、血；离为目、心、上焦；艮为手、鼻、背；兑为口、喉、肺

**六爻法**：
- 以官鬼爻为疾病（病源），子孙爻为医药
- 自占病 → 以世爻为自身，官鬼克世则病重
- 子孙爻旺相发动 → 药有效、病可愈
- 官鬼爻入墓或受克 → 病将退；官鬼爻旺动 → 病势重
- 久病逢冲 → 危险（冲散元神）；近病逢冲 → 快愈（冲散病气）

---

## 六、官讼占（官司、纠纷）

**梅花易数法**：
- 体克用 → 我胜，官司能赢
- 用克体 → 对方胜，宜退让和解
- 体用比和 → 宜和解，私了最好
- 乾为公门、官方；离为文书、证据；坎为险陷

**六爻法**：
- 以官鬼爻为官方、官府，父母爻为文书诉状
- 世爻为自己，应爻为对方
- 世克应 → 我胜；应克世 → 对方胜
- 官鬼爻生世 → 官方偏向我；官鬼爻克世 → 官方对我不利
- 子孙爻旺动 → 官司可解（子孙制官鬼）

---

## 七、出行占

**梅花易数法**：
- 体克用或体用比和 → 出行顺利
- 用克体 → 出行有险，不宜远行
- 坎为水 → 宜涉舟；震巽为木 → 宜东方；乾兑为金 → 宜西方
- 互见坎 → 防险陷；互见兑 → 防口舌；互见离 → 利文书之行

**六爻法**：
- 自占出行 → 以世爻为自身
- 官鬼爻旺动 → 途中有阻滞或祸患
- 兄弟爻旺动 → 途中有破财
- 子孙爻持世 → 一路平安
- 应爻为目的地，应生世则所到之处顺利

---

## 八、失物占

**梅花易数法**：
- 体克用 → 物可寻回
- 用克体 → 物难寻
- 用生体 → 物易寻回
- 乾兑 → 金属物品、在西北或西方
- 坤艮 → 土中、地下、在西南或东北
- 离 → 炉灶、火处、在南方
- 坎 → 水边、低洼处、在北方
- 震巽 → 木器中、树下、在东方或东南
- 变卦为体卦所生 → 物在附近；变卦克体卦 → 物已被毁或被盗

**六爻法**：
- 以妻财爻（财物）或父母爻（文书）为用神，看用神所在爻位和地支推断方位
- 用神旺相且与世爻相生合 → 可寻回
- 用神空亡或入墓 → 难寻
- 官鬼爻持世 → 被盗
- 用神合日辰 → 物未远走

---

## 九、行人占（占在外之人何时归）

**梅花易数法**：
- 用生体 → 归期近
- 体生用 → 归期远
- 体用比和 → 归期已定
- 用克体 → 暂时不归或有障碍

**六爻法**：
- 占行人 → 以用神（占何人取何六亲）为主
- 用神旺相 → 人平安；用神休囚 → 在外不顺
- 用神发动 → 已在归途；用神安静 → 尚未动身
- 用神与世爻相合 → 即将归来
- 父母爻代表书信音讯 → 父母爻动则有消息来

---

## 十、家宅占

**梅花易数法**：
- 体卦旺相 → 家宅安稳
- 用克体 → 家宅不安，有隐患
- 乾兑 → 注意金属伤、西北/西边有损
- 坤艮 → 注意土石问题
- 震巽 → 注意虚惊、木器损坏
- 离 → 防火灾；坎 → 防水患、偷盗

**六爻法**：
- 以父母爻为宅舍，二爻为宅，五爻为人
- 官鬼爻在二爻 → 宅中有祟或不安
- 子孙爻旺相 → 家宅平安
- 兄弟爻旺动 → 家人不睦或破财
- 妻财爻旺相 → 家业丰隆

---

## 十一、生产占（问怀孕、生育）

**梅花易数法**：
- 体卦旺相、用卦生体 → 生产顺利
- 体克用 → 稍有波折但可顺产
- 用克体 → 难产，需特别注意
- 阳爻多 → 可能生男；阴爻多 → 可能生女

**六爻法**：
- 以子孙爻为胎儿，妻财爻为产妇
- 子孙爻旺相 → 胎儿健康
- 胎爻（子孙爻所临地支的长生位）受冲 → 注意流产
- 兄弟爻旺动 → 生产时多消耗
- 官鬼爻动 → 生产中可能有险

---

## 使用策略

| 占问类型 | 优先用何法 | 关键判断点 |
|---------|-----------|-----------|
| 天时 | 梅花易数 + 八卦类象 | 离坎二卦的有无 |
| 婚姻 | 梅花易数 + 六爻用神 | 体用生克 + 世应合冲 |
| 求财 | 六爻用神为主 | 妻财爻 + 子孙爻(元神) + 兄弟爻(忌神) |
| 求名 | 六爻用神为主 | 官鬼爻 + 父母爻 |
| 疾病 | 梅花易数(定部位) + 六爻(定吉凶) | 官鬼爻 + 子孙爻 |
| 官讼 | 六爻用神为主 | 世应生克 + 官鬼爻倾向 |
| 出行 | 梅花易数体用 + 六爻 | 官鬼爻(险阻) + 子孙爻(平安) |
| 失物 | 梅花易数方位 + 六爻用神 | 用神旺衰空亡 |
| 行人 | 六爻用神为主 | 用神动静 + 世应合冲 |
| 家宅 | 梅花易数体用 + 六爻 | 二爻(宅) + 五爻(人) |
| 生产 | 六爻用神为主 | 子孙爻 + 胎爻 |

---

> **通用口诀**（节选《诸事响应歌》）：
> 「大抵体宜用卦生，旺相谋为终有益。比和为吉克为凶，生用亦为凶兆矣。」
> 意为：体卦宜被用卦所生，旺相时做事有利。比和为吉，克体为凶，体生用也是凶兆（耗泄）。
