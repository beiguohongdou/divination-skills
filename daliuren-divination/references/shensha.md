# 大六壬 · 常用神煞表

> 大六壬神煞众多，本表提取最常用者。神煞以日干、日支、月建、年命为参照定其位置。
> 来源：六壬大全、大六壬心镜

---

## 一、以日干为参照的神煞

| 神煞 | 取法 | 含义 |
|------|------|------|
| 禄神 | 日干之禄（甲寅乙卯丙戊巳丁己午庚申辛酉壬亥癸子） | 财禄、薪水、安稳 |
| 羊刃 | 禄前一位（甲卯乙寅丙戊午丁己巳庚酉辛申壬子癸亥） | 刚猛、易折、血光 |
| 文昌 | 日干之文昌位 | 文书、考试、学术 |
| 学堂 | 日干之长生位 | 求学、拜师 |

---

## 二、以日支为参照的神煞

| 神煞 | 取法 | 含义 |
|------|------|------|
| 驿马 | 寅午戌在申、申子辰在寅、巳酉丑在亥、亥卯未在巳 | 出行、变动、奔波 |
| 桃花 | 寅午戌在卯、申子辰在酉、巳酉丑在午、亥卯未在子 | 感情、艳遇、酒色 |
| 墓神 | 日支五行之墓（木在未、火在戌、金在丑、水土在辰） | 阻滞、隐藏、终结 |
| 华盖 | 寅午戌在戌、申子辰在辰、巳酉丑在丑、亥卯未在未 | 孤独、清高、学术 |

---

## 三、以月建为参照的神煞

| 神煞 | 取法 | 含义 |
|------|------|------|
| 天喜 | 正月在戌、二月在亥...顺行十二宫 | 喜庆、婚嫁 |
| 天赦 | 春戊寅、夏甲午、秋戊申、冬甲子 | 赦免、解厄 |
| 月破 | 与月建对冲之支 | 破损、无力 |
| 月德 | 寅午戌在丙、亥卯未在甲... | 贵人相助、消灾 |

---

## 四、以年命（出生年/属相）为参照的神煞

| 神煞 | 取法 | 含义 |
|------|------|------|
| 本命 | 出生年之地支 | 自己的根基 |
| 行年 | 本命起一岁，顺数至当年岁数所在 | 当年运势焦点 |
| 太岁 | 当年地支 | 流年主宰，不可犯 |

---

## 五、神煞快速判断

| 神煞组合 | 含义 |
|---------|------|
| 驿马 + 青龙 | 出行有喜，升迁出差 |
| 驿马 + 白虎 | 出行有险，奔波劳苦 |
| 桃花 + 六合 | 感情和合，姻缘动 |
| 桃花 + 玄武 | 暗昧偷情，非正缘 |
| 禄神 + 贵人 | 事业得贵，升职加薪 |
| 禄神 + 天空 | 事业遇空，虚有其位 |
| 墓神 + 螣蛇 | 阴事纠缠，噩梦惊扰 |
| 华盖 + 青龙 | 学术有成，文章出众 |
| 天喜 + 六合 | 婚庆之喜 |
| 天喜 + 天后 | 婚姻大吉 |
| 月破 + 白虎 | 月内凶灾 |
| 太岁临干 | 有流年大事发生 |
