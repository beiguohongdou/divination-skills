## 二、地支六合与六冲

**六合**（相合为吉，代表和谐、合作、助力）：
子丑合、寅亥合、卯戌合、辰酉合、巳申合、午未合

**六冲**（相冲为凶，代表冲突、变动、分离）：
子午冲、丑未冲、寅申冲、卯酉冲、辰戌冲、巳亥冲

> 生合则吉，冲克则凶。世应之间尤其看重合冲关系。

## 四、装世应口诀

> **八宫之卦世当六，以下初爻轮上装。**
> **游魂之卦四爻立，归魂八卦三爻详。**

- 本宫卦（纯卦）：世在**上爻**（第六爻）
- 一世卦：世在**初爻**
- 二世卦：世在**二爻**
- 三世卦：世在**三爻**
- 四世卦：世在**四爻**
- 五世卦：世在**五爻**
- 游魂卦：世在**四爻**
- 归魂卦：世在**三爻**

**应爻**与世爻相隔二爻：世在初则应在四，世在二则应在五，世在三则应在上，世在四则应在初，世在五则应在二，世在上则应在三。

## 六、装六亲

六亲按本宫卦的五行属性来确定：

1. **本宫五行**：乾兑金、坤艮土、震巽木、坎水、离火
2. **各爻地支的五行** → 与本宫五行比较：
   - 同我者：**兄弟**
   - 生我者：**父母**
   - 我生者：**子孙**
   - 克我者：**官鬼**
   - 我克者：**妻财**

例：乾宫属金，乾为天各爻地支为子(水)、寅(木)、辰(土)、午(火)、申(金)、戌(土)
- 子水：金生水 → **子孙**
- 寅木：金克木 → **妻财**
- 辰土：土生金 → **父母**
- 午火：火克金 → **官鬼**
- 申金：同金 → **兄弟**
- 戌土：土生金 → **父母**

> 六神为辅助信息，不单独断吉凶。用时结合爻的旺衰和动变参考。

## 八、用神取用

根据所问之事，确定用神所在：

| 事类 | 用神 |
|------|------|
| 占自己 | 世爻 |
| 占父母、师长、文书、房屋 | 父母爻 |
| 占兄弟、朋友、同辈 | 兄弟爻 |
| 占子孙、晚辈、医药、宠物 | 子孙爻 |
| 占妻妾、财物、货物 | 妻财爻 |
| 占官运、功名、丈夫（女占）、鬼神 | 官鬼爻 |
| 占他人在外、无关系之人 | 应爻 |

## 九、世应用法

- **世爻**：代表占问者自身
- **应爻**：代表所问之人、事、环境
- 世生应 → 自己付出较多
- 应生世 → 他人或环境有利
- 世克应 → 自己主导，可成
- 应克世 → 外力阻碍，困难
- 世应比和 → 双方和谐，顺利
- **世空** → 自己心不诚或事难成
- **应空** → 对方无意或事无着落
