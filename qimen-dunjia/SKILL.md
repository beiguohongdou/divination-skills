# 奇门遁甲

奇门遁甲，三式之首（奇门、六壬、太乙），古代帝王之学。核心是**时空决策模型**：不是告诉你「能不能」，而是告诉你「什么时候在哪个方位做什么事」。

不同于六爻（随即起卦）和大六壬（按时间定盘），奇门遁甲的排盘按**节气+时辰**确定阴阳遁局数，排出一个 3×3 的九宫格，每个宫位各有吉凶含义。

## 占卜规范

### 道德原则

奇门遁甲为三式之首，帝王之学，当守以下规范：

1. **不过度依赖**：占卜是参考，不是人生的GPS导航
2. **不替代专业决策**：医疗、法律、财务问题应咨询专业人士
3. **不恐吓用户**：重在分析趋势和提供化解方向，不说「大凶」「必死」
4. **尊重隐私**：占卜内容保密，不外泄
5. **诚心专一**：奇门按时辰节气排盘，需心诚意正，否则局象不验

### 禁忌原则

#### 不占之事

- **不占生死寿数**：「我什么时候会死」——此类问题不占
- **不占已知结果**：明知故问、试探——无诚意则局不验
- **不占损人害己**：如何害人、如何违法——违道德伦理之事不占
- **不占戏谑玩闹**：「随便起一局看看」——无具体事由则局象空泛

#### 不占之时

- **心不诚时不占**：心存疑虑、半信半疑——局象不准
- **酒后失神时不占**：神志不清则无法诚心感应
- **一事反复占**：说明内心已有答案——「再三则渎，渎则不告」

#### 解读禁忌

- **不说绝对化措辞**：「必死」「必败」「必成」——奇门讲时空趋势，不测必然
- **不用恐吓性语言**：「血光之灾」「家破人亡」——即使局象不吉，也要指出化解方向
- **不替用户做决定**：提供分析和方位建议，决定权在用户
- **不泄露天机过度**：点到为止

## 起局前引导

奇门遁甲需要**精确的时间和明确的方位/问题**：

> 「奇门遁甲按时辰和节气排盘。你问的是哪个方面的事？比如事业、感情、出行，还是看今天某个时段的运势？另外有没有特别关心的方位（如东南、西北）？如果没有特别时间，我用当前时辰起局。」
>
> **方位判断说明**：奇门方位判断需要知道你的大概位置和目标的方向关系。方便的话告诉我你在哪个城市、目标大致在哪个方位即可（如："我在广州，公司在成都西北方向"）。城市级别就足够判断，不需要精确到街道。如果不方便提供，我不做方位解读，只论格局吉凶。

## 排盘方法

### 第一步：定局（阴阳遁十八局）

按**节气**确定用阴遁还是阳遁，以及局数：

| 节气范围 | 遁法 | 局数规则 |
|---------|------|---------|
| 冬至→夏至 | 阳遁 | 甲子戊从1宫到9宫（阳遁1-9局） |
| 夏至→冬至 | 阴遁 | 甲子戊从9宫到1宫（阴遁1-9局） |

**定局口诀**：冬至一七四、小寒二八五、大寒三九六、立春八五二、雨水九六三、惊蛰一七四、春分三九六、清明四一七、谷雨五二八、立夏四一七、小满五二八、芒种六三九；夏至九三六、小暑八二五、大暑七一四、立秋二五八、处暑一四七、白露九三六、秋分七一四、寒露六九三、霜降五八二、立冬六九三、小雪五八二、大雪四七一。

✅ **自检**：
- 当前节气是否在冬至→夏至之间（阳遁）还是夏至→冬至之间（阴遁）？
- 口诀中当前节气对应的局数是否合理？

### 第二步：排地盘（三奇六仪入九宫）

阳遁顺排六仪（戊己庚辛壬癸）顺排三奇（丁丙乙）；阴遁逆排六仪，顺排三奇。

**九宫顺序固定**：1坎→2坤→3震→4巽→5中→6乾→7兑→8艮→9离

✅ **自检**：
- 阳遁是否从坎1宫开始顺排？阴遁是否从离9宫开始逆排？
- 三奇六仪的顺序是否与遁法匹配？
- 九宫顺序是 1→9，不要跳宫或漏宫？

### 第三步：排天盘（时干加临）

以时干所在之宫为参照，将值符（旬首对应的星）加于时干之上，顺转排布九星。

✅ **自检**：
- 时干是否正确？
- 值符星是否加到了时干对应的宫位？
- 九星是否按顺序顺转排布？

### 第四步：排八门

以值使（旬首对应的门）加于时支之上，顺转排布八门。

✅ **自检**：
- 值使门是否加到了时支对应的宫位？
- 八门是否按顺序顺转？
- 八门的数量是8个，不要有遗漏？

### 第五步：排八神

阳遁顺排、阴遁逆排。顺序：值符、螣蛇、太阴、六合、白虎、玄武、九地、九天。

✅ **自检**：
- 阳遁是否顺排？阴遁是否逆排？
- 八神的顺序是否从值符开始？
- 八神数量为8个，是否有遗漏？

### 第六步：排九星

值符星加于时干之上，顺转排布。

✅ **自检**：
- 九星是否按顺转排布？
- 九星数量为9个，是否有遗漏？

## 断局方法

### 核心思路

| 步骤 | 内容 |
|------|------|
| 看日干落宫 | 代表问事人自身的状态和环境 |
| 看时干落宫 | 代表所问之事的状态 |
| 看用神落宫 | 根据所问之事取用神宫位 |
| 看八门吉凶 | 所落之宫门是吉是凶、是否有迫/制 |
| 看九星旺衰 | 星是否当令（与我生、生我、克我、我克的关系） |
| 看八神辅助 | 神煞增损吉凶 |
| 看格局克应 | 十干克应、八门克应等具体组合含义 |

### 解读框架

```
## 🌐 起局信息
- 时间：某年某月某日某时
- 节气：某某
- 遁法：阳/阴遁 N 局

## 📐 九宫格局
（九宫格ASCII图，标注各宫的三奇六仪、八门、九星、八神）

## 🎯 用神分析
- 日干落宫：[宫位] [门] [星] [神] → 问事人状态
- 时干落宫：[宫位] [门] [星] [神] → 所问之事状态
- 用神落宫：[宫位] → 核心判断

## 🔍 格局克应
（各宫出现的特殊格局及含义）

## 🧭 时空建议
- 有利方位：[哪几个宫位吉利]
- 有利时段：[哪个时段行动]
- 忌避方位：[哪几个宫位不利]
```

### 解读原则

- **先看大局后看细节**：八门九星八神的整体气势 > 单个格局
- **门迫宫凶、宫迫门亦凶**：吉门临迫宫则减吉，凶门临迫宫则增凶
- **三奇得使最吉**：乙丙丁三奇加临值使门为最佳格局
- **不恐吓**：奇门格局繁多且多凶名，重在指出理解和化解方向
- **有时空维度**：与六爻/大六壬最大的区别是给出**具体方位**和**时段建议**

### 多体系冲突处理

当梅花易数、大六壬、奇门遁甲三体系同时用于同一问题时，按以下三层结构呈现结果：

**第一层 · 总纲 —— 梅花易数**
体用生克直接输出吉凶趋势，作为结论主调。

**第二层 · 过程 —— 大六壬**
三传（初传→中传→末传）描述事情的发展演变，修正或细化梅花的总纲判断。

**第三层 · 行动 —— 奇门遁甲（自身）**
提供具体的时空建议（方位、时机、注意点）。当自身与前两层结论矛盾时，优先检查排盘是否正确；若排盘无误但结论仍不协调，则侧重于提供"注意事项"而非"吉利判断"。

> 若三者结论矛盾，以梅花易数的体用生克为主纲解读，大六壬描述过程变化，奇门遁甲补充行动建议，避免将互相矛盾的原始结果直接堆给用户。

### 反馈闭环

用户事后反馈时（「上次那局准了」「不准，实际情况是...」），将反馈追加到对应记录的 `feedback` 字段，并根据实际结果更新 `accuracy`（1-5分，5为完全准确）。

### 积累分析

定期（每10局或用户要求时）读取日志，分析：
- 总体准确率
- 哪类问题断得准、哪类不准
- 各格局的准确率差异
- 八门九星八神判断的偏差规律

这些分析结果可以反哺后续的解读权重。
