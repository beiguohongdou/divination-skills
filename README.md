# 三式占卜 Skills

梅花易数/六爻、大六壬、奇门遁甲 三套 AI Agent 占卜技能。

## 包含

| Skill | 体系 | 参考文件 |
|-------|------|---------|
| `meihua-yijing-divination` | 梅花易数 + 六爻 | 8 个（黄金策、增删卜易、纳甲八宫、万物类象等） |
| `daliuren-divination` | 大六壬神课 | 6 个（毕法赋100条、邵公断案、天地盘等） |
| `qimen-dunjia` | 奇门遁甲 | 6 个（十干克应、八门九星、格局克应等） |

## 安装

复制对应 skill 目录到你的 Agent 技能目录（如 `~/.hanako/skills/` 或 `~/.agents/skills/`）。

## 数据来源

古籍原文来自 [luckclub.cn](https://www.luckclub.cn)，PDF 仅供文化学习研究。
