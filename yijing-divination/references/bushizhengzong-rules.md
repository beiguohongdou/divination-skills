## 二、世应论用神

### 基本原则

- **世为自己**，**应为他人**
- 世应相生相合：宾主相投
- 世应相克相冲：两情不睦

### 世爻为用的情况

- 占自己疾病
- 问寿数
- 问出行吉凶
- 诸凡损益自身者

### 应爻为用的情况

- 占无尊卑之称呼、未曾深交之朋友
- 九流术士、仇人、敌国
- 指实某处地头（此山、此水、此寺、此塔等）

### 特殊用法

- 占自己有一地可造坟否：世为穴场，应为对案
- 将买他人之地而欲造坟：应作穴场，世是我家

## 三、原忌仇神论

### 基本概念

- **原神**：生用神之爻（用神的助力）
- **忌神**：克用神之爻（用神的阻力）
- **仇神**：克原神生忌神之爻（间接破坏）

### 原神断法

1. 用神旬空、月破、衰弱、伏藏不现
   - 得原神动来生之，或日辰月建作原神生之
   - 必待用神出旬出破得令值日，所求必遂

2. 用神旺相，原神休囚不动
   - 或动而变墓、变绝、变克、变破、变退
   - 或被日辰月建克制
   - 皆不能生用，用神根蒂被伤，反有损

### 忌神断法

1. 忌神动来克用，用爻出现不空 → 受克

2. 卦中又动出一爻原神生用
   - 忌神反生原神，是名「贪生忘克」
   - 用神根蒂深固，其吉更倍

3. 忌神独发，用爻旬空 → 避空，不为害
4. 忌神独发，用爻伏藏不现 → 避凶，不为害
5. 忌神独发，月建日辰生用 → 得救，仍为吉兆

6. 忌神变回头之克，或日辰月建克冲之，或动爻制忌之
   - 谓之「贼欲害我是贼先受害」，我无伤

7. 日辰月建生扶忌神，或忌神叠叠克用
   - 即使用神避空伏藏者，至出空出透时便受其毒，难免其灾

## 四、用神旺衰判断

### 旺相休囚死

| 状态 | 含义 | 力量 |
|------|------|------|
| 旺 | 当令，得时 | 最强 |
| 相 | 次旺，将旺 | 次强 |
| 休 | 休息，不当令 | 中等 |
| 囚 | 被困，受制 | 较弱 |
| 死 | 最弱，无气 | 最弱 |

### 判断方法

- **看月建**：月建生扶为旺，月建克泄为衰
- **看日辰**：日辰生扶为旺，日辰克泄为衰
- **看动爻**：动爻生扶为旺，动爻克泄为衰

## 五、旬空详解

### 旬空表（按日干支查）

| 旬首 | 旬空地支 |
|------|----------|
| 甲子旬 | 戌、亥 |
| 甲戌旬 | 申、酉 |
| 甲申旬 | 午、未 |
| 甲午旬 | 辰、巳 |
| 甲辰旬 | 寅、卯 |
| 甲寅旬 | 子、丑 |

### 断法

- **用神旬空**：事不落实，心中无底，待出旬（旬空地支值日）应事
- **忌神旬空**：忌神无力，反而有利
- **世爻旬空**：自己心虚或诚意不足
- **应爻旬空**：对方无意或事情落空
- **动爻旬空**：动不为空，出旬即应；但动而化空则减力
- **静爻旬空**：真空无力，需待填实

### 出空与填实

- **出空**：旬空地支在日辰上出现之日（如午未空，到午日或未日出空）
- **填实**：旬空地支在月建上出现（如午空，月建为午月则填实）

## 六、暗动与日破

### 暗动

**定义**：静爻被日辰冲（六冲关系），称为「暗动」。暗动之爻虽本卦中为静爻，但在实际断卦中应视为有动爻之效。

**判定条件**：
1. 该爻必须是静爻（非老阴/老阳）
2. 该爻地支与当日地支形成六冲关系
3. 该爻不处于旬空状态（旬空被冲为「冲空」，不作暗动论）

**暗动的效力**：
- 暗动之爻可以生克其他爻
- 暗动之爻可以作为用神、忌神的补充力量
- 暗动是「暗中发动」，事情在不知不觉中发生变化
- 暗动持续时间短，通常在当日或短期应验

**与明动的区别**：
- 明动（老阴/老阳）：变化明确，影响持久
- 暗动（日冲静爻）：变化隐蔽，影响短暂

### 日破

**定义**：爻的地支与当日地支相冲，且该爻为衰弱状态（休囚无生扶），称为「日破」。

**日破与暗动的区别**：
- 静爻旺相被日冲 → 暗动（有力）
- 静爻衰弱被日冲 → 日破（无力）
- 关键在于爻本身的旺衰状态

**断法**：
- 用神日破 → 事难成，需待出破之日
- 忌神日破 → 忌神无力，反而有利
- 日破之爻逢生扶 → 可解破

## 七、反吟与伏吟

### 反吟

**定义**：变卦与本卦的地支完全对冲（六冲关系）。

**断法**：反吟主反复、变化无常。事情会来回折腾，难以稳定。
- 占病反吟主反复发作
- 占事反吟主办成又变

**简化判断**：变卦与本卦卦名互为「对冲卦」（如乾变坤、坎变离、震变巽），即为反吟。

### 伏吟

**定义**：变卦与本卦的地支完全相同（无变化）。

**断法**：伏吟主停滞、原地踏步。事情没有进展，想动动不了。
- 占出行伏吟主去不成
- 占病伏吟主缠绵难愈

**简化判断**：变卦与本卦卦名相同（如乾变乾、坎变坎），即为伏吟。

### 组合判断

若卦中既有反吟又有伏吟（部分爻反吟、部分爻伏吟），主事情先反复后停滞，或反复中夹杂停滞。

## 八、进神退神

### 进神

**定义**：动爻变出之爻与本爻同五行，且地支顺序前进（如寅变卯、申变酉）。

**断法**：进神主事物向前发展、力量增强。
- 用神化进神：事情越来越好
- 忌神化进神：阻碍越来越大

### 退神

**定义**：动爻变出之爻与本爻同五行，且地支顺序后退（如卯变寅、酉变申）。

**断法**：退神主事物后退、力量减弱。
- 用神化退神：事情越来越差
- 忌神化退神：阻碍越来越小

## 九、飞伏神

### 基本概念

- **飞神**：卦中出现之爻
- **伏神**：卦中不出现之爻，伏于某爻之下

### 取用规则

当用神不现（卦中无用神）时，需取伏神：
1. 先看本宫首卦（八纯卦），找到用神
2. 看用神伏于何爻之下
3. 该爻为飞神，用神为伏神

### 断法

- 伏神得飞神生扶 → 伏神有力
- 伏神被飞神克制 → 伏神无力
- 伏神旬空、月破、衰弱 → 伏神无用
- 伏神旺相、得日辰生扶 → 伏神可用

## 十、卦身

### 定义

卦身是卦中代表所占之事的爻，不同于世爻（代表自己）和应爻（代表他人）。

### 取法

「阳世则从子月起，阴世还从午月生」：
- 阳爻持世：从子月起，数到月建所在之地支
- 阴爻持世：从午月起，数到月建所在之地支

### 断法

- 卦身为所占事之体
- 六爻中有两爻出现卦身 → 鸳鸯求事，或事于两处
- 卦身带兄弟 → 与人同谋
- 卦身克世 → 事寻我吉
- 世克卦身 → 凶
- 卦身生合世爻 → 更吉
