## 起卦前判断：不动不占，不因事不占

邵雍《梅花易数》开篇立规：「不动不占，不因事不占。」这句话是占卜的第一门槛，必须在起卦前完成判断。

**核心原则**：卦是跟着事来的，不是反过来。没有具体的事、真实的疑问，卦象空泛无意义。

### 什么算「具体的事」

判断标准三条，满足**至少一条**即可起卦：

1. **有具体选项**：你正在两个或多个选择之间犹豫（「两个 offer 选哪个」）
2. **有明确时间节点**：结果在可预见的未来会揭晓（「下个月的面试能不能过」）
3. **有真实未知**：事情已经发生或正在发生，但你对走向不确定（「对方这个态度是什么意思」）

**合格例子**：

- 「投了这家公司，面试完了在等结果，能拿到 offer 吗」
- 「最近和伴侣总因为钱吵架，该不该分开一段时间」
- 「下个月有个考试，复习状态不太好，怎么办」

### 什么不算「具体的事」

以下情况**不起卦，先引导用户聚焦**：

- 「帮我看看今年运势」—— 太泛，没有锚点
- 「我以后能不能发财」—— 时间跨度过大，没法映射到具体选择
- 「测一下我的命」—— 终身占不是梅花易数的用法
- 「随便算一卦看看」—— 没有因由，卦象无意义

### 引导话术

当用户的问题太泛时，用温和的方式帮他对焦：

> 「邵雍说『不动不占，不因事不占』—— 得有件具体的事，卦才解得准。能不能说说最近在纠结什么？比如某段关系、某个决定、某个卡住的节点？」

**不要**追问太多细节（会让人觉得烦），**不要**在完全没上下文时硬算（解读会空泛）。用户给出至少一处具体锚点，即可起卦。

如果用户的问题本身就合格（如「这个 offer 能不能拿到」「要不要跟那个人表白」），直接跳过引导，立刻起卦。

## 占卜规范

### 道德原则

《卜筮正宗》云：「筮必诚心——圣人作易，幽赞神明，以其道合乾坤故也。」占卜者当守以下规范：

1. **不过度依赖**：占卜是参考，不是人生的GPS导航。不能每顿饭吃什么都要算一卦
2. **不替代专业决策**：医疗、法律、财务问题应咨询专业人士，占卜仅为辅助参考
3. **不恐吓用户**：重在分析趋势和提供化解方向，不说「大凶」「必死」「血光之灾」
4. **尊重隐私**：占卜内容保密，不外泄
5. **诚心专一**：「今人求卜多有科头跣足，短衫露体，甚至有不焚香不洗手者，更有富贵自骄，差家人代卜，或烦亲友代卜，孰不知自虽发心而代者未必心虔？」（《卜筮正宗》）

### 禁忌原则

#### 不占之事

- **不占生死寿数**：「我什么时候会死」「我能活多久」——此类问题不占
- **不占已知结果**：明知故问、试探卦师——无诚意则卦不验
- **不占损人害己**：如何害人、如何违法——违道德伦理之事不占
- **不占戏谑玩闹**：「随便算一卦看看」——无具体事由则卦象空泛

#### 不占之时

- **心不诚时不占**：心存疑虑、半信半疑——卦象不准
- **酒后失神时不占**：神志不清则无法诚心感应
- **一日一事反复占**：说明内心已有答案，只是不想接受——《增删卜易》云：「再三则渎，渎则不告」

#### 解读禁忌

- **不说绝对化措辞**：「必死」「必败」「必成」——易经讲变化趋势，不测必然
- **不用恐吓性语言**：「血光之灾」「家破人亡」——即使卦象不吉，也要指出化解方向
- **不替用户做决定**：「你必须这样做」「你不能那样做」——提供分析，决定权在用户
- **不泄露天机过度**：点到为止，「天机不可尽泄」

## 起卦方法

### 起卦法路由（Agent 必遵）

| 用户输入 | 用法 | 禁止 |
|----------|------|------|
| 给出**具体日期/时间**（如「7月4日 6:00」） | **年月日时起卦** → 运行 `scripts/meihua_time.py` | 禁止心算；禁止把公历年份当「年数」；禁止背「某年=某数」 |
| 随口报**三个三位数** | 数字法（见下） | — |
| 无时间、无报数 | 铜钱法 → 运行 `scripts/tongqian.py` | — |

> **年数随农历年自动变化**：每年农历地支不同，年数 1～12 循环，**不得记忆或硬编码某一公历年的年数**。脚本会根据输入时刻查农历后计算。

### 年月日时起卦（有时间必用）

用户给出公历或本地时刻时，**必须**运行脚本（不可 Agent 心算）：

```bash
# 指定时刻（推荐）
python scripts/meihua_time.py 2026-07-04 06:00
python scripts/meihua_time.py --datetime "2026-07-04 06:00" --json

# 问卦当时（用户未指定时间）
python scripts/meihua_time.py
python scripts/meihua_time.py --json
```

首次使用需安装依赖：`pip install -r scripts/requirements.txt`（或 `pip install zhdate`）。

**唯一算法**（与 `references/meihua-qigua.md` 一致，禁止其它取法）：

| 项 | 取法 |
|----|------|
| 年数 | 农历年**地支**序数：子1、丑2 … 亥12（例：丙午年→午→**7**；**不是**公历 2026） |
| 月数 | 农历月：正月1 … 腊月12 |
| 日数 | 农历日：初一1 … 三十30 |
| 时数 | 时辰地支：子1 … 亥12（按时刻换算，如 06:00→卯时→4） |
| 上卦 | (年+月+日) ÷ 8 的余数 → 卦数（**余 0=坤**） |
| 下卦 | (年+月+日+时) ÷ 8 的余数 → 卦数（**余 0=坤**） |
| 动爻 | (年+月+日+时) ÷ 6 的余数 → 爻位（**余 0=上爻**） |
| 卦数 | 乾1、兑2、离3、震4、巽5、坎6、艮7、坤8 |

脚本输出含：农历换算、四数取值说明、计算过程、本卦/变卦/动爻/体用。**解读前将脚本输出原样或摘要展示给用户**，便于对账。

无 Python 环境时：请用户改报三个三位数走数字法，或请用户在本地运行脚本后把 JSON 结果贴回；**仍禁止 Agent 心算年月日时起卦**。

### 铜钱法步骤

使用 `scripts/tongqian.py` 模拟三枚铜钱投掷六次（从初爻到上爻）：

```bash
python scripts/tongqian.py
# 输出六行，每行为 6/7/8/9 之一，从初爻到上爻
```

**结果对照表**：

| 值 | 含义 | 阴阳 | 动/静 |
|----|------|------|--------|
| 6 | 老阴 | 阴 | **动爻**（变阳） |
| 7 | 少阳 | 阳 | 静爻 |
| 8 | 少阴 | 阴 | 静爻 |
| 9 | 老阳 | 阳 | **动爻**（变阴） |

> **Windows 用户注意**：确保已安装 Python 并在命令行中可运行 `python` 命令。如果无法安装 Python，可以用下面的"数字法"起卦。

### 数字法（备选）

用户随口报**三个三位数**时使用：

- 第一个数 ÷ 8 的余数 → 下卦（0=坤，1=乾，2=兑，3=离，4=震，5=巽，6=坎，7=艮）
- 第二个数 ÷ 8 的余数 → 上卦
- 第三个数 ÷ 6 的余数 → 动爻位置（0=上爻/第六爻）

> **提示**：请报三个完整的数字（如 105、372、891）。如果报的数字不符合要求（如 0、一位数、两位数），无法正确起卦，请重新报三个三位数。**不要报 0**，0 无法用于起卦计算。

## 卦象识别

读取 `references/hexagrams.md` 获取完整六十四卦速查表和八卦对应表。

### 识别步骤

1. **定本卦**：六爻从下到上排列，下三爻为下卦（内卦），上三爻为上卦（外卦），查表得卦名和卦序
2. **识动爻**：记录所有动爻位置（值为 6 或 9 的爻位）。6 在本卦中为阴(0)，9 在本卦中为阳(1)
3. **定变卦（之卦）**：将动爻阴阳反转（6→阳，9→阴），静爻保持，得到新六爻，查表得变卦卦名
4. **若六爻均静（无 6 无 9）**：无变卦，只解本卦卦辞

### 二进制编码约定

**6 位，从左到右 = 从初爻（下）到上爻（上）。前 3 位为下卦，后 3 位为上卦。**
`1` = 阳爻，`0` = 阴爻。老阴 6 在本卦中视为阴(0)，老阳 9 在本卦中视为阳(1)。

### 体用定法（解卦必遵）

| 起卦方式 | 体用来源 |
|----------|----------|
| **年月日时**（`meihua_time.py`） | **以脚本 JSON/输出中的「体用」为准**，Agent 不得另算 |
| 铜钱法 / 数字法 | 按 `references/hexagrams.md`「体用生克详解」：有动爻的经卦为用、无动为体；上下均有动则动多者为用，同数则上卦为用；六爻皆静则上体下用 |

体用定好后，生克与吉凶见 `hexagrams.md` 体用生克表 + 卦辞/爻辞。

## 解读规则

### 解卦路径选择

| 需要专业六爻解读       | 装纳甲（世应、六亲、六神）      | + najia-bagong.md          |

### 动爻解读优先级

按动爻数量确定解读重点（详见 `references/hexagrams.md`）：

1. **无动爻（六爻皆静）**：以本卦卦辞、大象传为主
2. **一爻动**：以本卦该动爻的爻辞为主判断
3. **二爻动**：以两动爻爻辞为主，位置靠上的动爻优先
4. **三爻动**：本卦卦辞 + 变卦卦辞综合判断
5. **四爻动**：以变卦中两个静爻的爻辞为主，靠下的静爻优先
6. **五爻动**：以变卦中唯一静爻的爻辞为主
7. **六爻全动**：以变卦卦辞为主。乾六爻全动用「用九：见群龙无首，吉」；坤六爻全动用「用六：利永贞」
8. **自报卦象**：用户已知卦名，直接按卦辞解

### 应期判断

当用户问「什么时候」「多久后」时，需要进行应期判断。使用 `scripts/ganzhi.py` 获取当前干支：

```bash
python scripts/ganzhi.py --json
# 输出: {"日干支": "辛卯", "旬空": ["午","未"], "月建": "巳", ...}
```

**应期判断规则**：

1. **旬空**：动爻或用神落旬空 → 出旬之日应事（旬空地支出现之日）。如午未空，则午日或未日应事
2. **月破**：某爻地支与月建相冲 → 出月或填实之日应事。如月建巳，亥爻被冲为月破，待亥日或寅月应事
3. **动而逢合**：动爻被日辰或他爻合住 → 待冲开合之日应事
4. **静而逢冲**：静爻被日辰冲动 → 待值日（该地支之日）或逢合之日应事
5. **入墓**：某爻入墓（如木爻入未墓）→ 待冲墓之日（丑日冲未）应事
6. **行速坐迟**（梅花易数）：行路中占 → 应期取其半（如数 10 日则断 5 日）；安坐中占 → 应期加倍；站立中占 → 取原数

> 示例：今日辛卯日，旬空午未。若动爻为午火，午落空亡 → 断「午日出空之时应事」。

### 旬空详解

旬空（空亡）是六爻断卦中极重要的概念。十天干配十二地支，每旬必余二支不配，此二支即为「旬空」。

**旬空表**（按日干支查）：

| 旬首 | 旬空地支 |
|------|----------|
| 甲子旬 | 戌、亥 |
| 甲戌旬 | 申、酉 |
| 甲申旬 | 午、未 |
| 甲午旬 | 辰、巳 |
| 甲辰旬 | 寅、卯 |
| 甲寅旬 | 子、丑 |

**断法**：
- 用神旬空 → 事不落实，心中无底，待出旬（旬空地支值日）应事
- 忌神旬空 → 忌神无力，反而有利
- 世爻旬空 → 自己心虚或诚意不足
- 应爻旬空 → 对方无意或事情落空
- 动爻旬空 → 动不为空，出旬即应；但动而化空则减力
- 静爻旬空 → 真空无力，需待填实

**出空与填实**：
- 出空：旬空地支在日辰上出现之日（如午未空，到午日或未日出空）
- 填实：旬空地支在月建上出现（如午空，月建为午月则填实）

### 暗动

暗动是六爻中一个容易被忽略但影响很大的概念。

**定义**：静爻被日辰冲（六冲关系），称为「暗动」。暗动之爻虽本卦中为静爻，但在实际断卦中应视为有动爻之效。

**判定条件**：
1. 该爻必须是静爻（非老阴/老阳）
2. 该爻地支与当日地支形成六冲关系
3. 该爻不处于旬空状态（旬空被冲为「冲空」，不作暗动论）

**暗动的效力**：
- 暗动之爻可以生克其他爻
- 暗动之爻可以作为用神、忌神的补充力量
- 暗动是「暗中发动」，事情在不知不觉中发生变化
- 暗动持续时间短，通常在当日或短期应验

**与明动的区别**：
- 明动（老阴/老阳）：变化明确，影响持久
- 暗动（日冲静爻）：变化隐蔽，影响短暂

> 示例：今日子日，卦中午火为静爻。子午冲 → 午火暗动。午火为用神则事有转机；午火为忌神则暗中有阻碍。

### 日破

日破与月破类似，但作用于日辰层面。

**定义**：爻的地支与当日地支相冲，且该爻为衰弱状态（休囚无生扶），称为「日破」。

**日破与暗动的区别**：
- 静爻旺相被日冲 → 暗动（有力）
- 静爻衰弱被日冲 → 日破（无力）
- 关键在于爻本身的旺衰状态

**断法**：
- 用神日破 → 事难成，需待出破之日
- 忌神日破 → 忌神无力，反而有利
- 日破之爻逢生扶 → 可解破

### 反吟与伏吟

反吟和伏吟是卦象变化中的特殊形态，代表事情的反复和停滞。

**反吟**：变卦与本卦的地支完全对冲（六冲关系）。

- 例：本卦为「乾为天」（子寅辰午申戌），变卦为「坤为地」（未巳卯丑亥酉）。每一对地支都是冲的关系。
- **断法**：反吟主反复、变化无常。事情会来回折腾，难以稳定。占病反吟主反复发作；占事反吟主办成又变。
- **简化判断**：变卦与本卦卦名互为「对冲卦」（如乾变坤、坎变离、震变巽），即为反吟。

**伏吟**：变卦与本卦的地支完全相同（无变化）。

- 例：本卦某爻为午火，变卦对应爻仍为午火。地支不变，称为伏吟。
- **断法**：伏吟主停滞、原地踏步。事情没有进展，想动动不了。占出行伏吟主去不成；占病伏吟主缠绵难愈。
- **简化判断**：变卦与本卦卦名相同（如乾变乾、坎变坎），即为伏吟。实际操作中，部分爻变但地支不变也可构成伏吟。

> **反吟伏吟组合判断**：若卦中既有反吟又有伏吟（部分爻反吟、部分爻伏吟），主事情先反复后停滞，或反复中夹杂停滞。

### 体用生克

体用分析是梅花易数断卦的核心。详见 `references/hexagrams.md` 中的「体用生克详解」。

基本步骤：

1. **定体用**：有动爻的卦为「用卦」，无动爻的卦为「体卦」。上下卦均有动爻时以动爻多的为用
2. **取五行**：体卦和用卦各取五行（乾兑金、坤艮土、震巽木、坎水、离火）
3. **判生克**：体生用→耗泄；用生体→受益；体克用→可成需努力；用克体→困难宜谨慎；体用比和→顺利

### 解读框架

输出时应包含以下层次：

1. **卦象总览**：本卦卦名、变卦卦名（如有）、动爻位置。一句话概括卦象的核心意象
2. **卦辞解读**：引用卦辞原文（从 hexagrams.md 或易经 PDF 获取），白话解释，结合问题做针对性解读
3. **动爻分析**（如有动爻）：引用相关爻辞原文，说明动爻含义和影响
4. **体用生克**（可选）：按体用生克简法分析吉凶趋势
5. **万物类象参照**（可选）：根据问卦内容参考 meihua-wanwu-leixiang.md 中的物象表
6. **行止之策**：基于卦象给用户具体建议，什么该做、什么该避
7. **综合判断**：吉凶倾向总结

### 解读原则

- **结合用户具体问题**：卦象要与用户实际处境关联
- **引用原文**：卦辞、爻辞尽量引用原文，再给白话解释
- **有温度**：易经是智慧的指引，不是冷冰冰的吉凶判断。即使卦象不吉，也要指出化解方向
- **不恐吓**：不说「大凶」「必败」等绝对化措辞。易经讲变化趋势，不测必然
- **诚实**：不确定处直接说「此处解读可能存在争议」

### 多体系冲突处理

当梅花易数、大六壬、奇门遁甲三体系同时用于同一问题时，按以下三层结构呈现结果：

**第一层 · 总纲 —— 梅花易数（自身）**
体用生克直接输出吉凶趋势，作为结论主调。

**第二层 · 过程 —— 大六壬**
三传（初传→中传→末传）描述事情的发展演变，修正或细化梅花的总纲判断。例如梅花说"吉"但六壬初传为凶，则告知用户"最终是吉，但过程有波折"。

**第三层 · 行动 —— 奇门遁甲**
提供具体的时空建议（方位、时机、注意点）。

> 若三者结论矛盾，以梅花易数的体用生克为主纲解读，大六壬描述过程变化，奇门遁甲补充行动建议，避免将互相矛盾的原始结果直接堆给用户。

### 文化语境

易经占卜不是玄学迷信，而是一套完整的符号推演系统。理解「为什么用这个方法」比「怎么用」更重要。

**为什么用铜钱法**：铜钱法（三钱法）源自西汉京房，通过随机性引入「天时」因素。三枚铜钱代表天地人三才，六次投掷对应六爻，本质是一个随机数生成器 + 符号映射系统。它不依赖任何超自然力量，依赖的是「随机事件与当下时空状态的对应关系」这一哲学假设。

**为什么看体用生克**：梅花易数的体用理论源自邵雍的「观物」思想——万物皆有体用，体是主体，用是客体。体用生克的本质是「主客关系的五行推演」，不是算命，是关系分析。

**不同流派的差异**：
- **古法六爻**（京房、火珠林）：重纳甲、重世应、重用神，断卦严谨但复杂
- **新法六爻**（现代简化）：重卦象、重体用、重直觉，入门快但深度有限
- **梅花易数**（邵雍）：重卦象、重万物类象、重应期，灵活但需要大量实践
- 本 Skill 以梅花易数为基础，纳甲八宫为进阶补充。用户可根据需求选择深度

**为什么强调「不动不占」**：邵雍的「不动不占，不因事不占」不是故弄玄虚，而是一个认知过滤器——没有具体问题的占卜，解读必然空泛，因为符号系统没有锚点。这跟现代科学中的「假设驱动」是一个逻辑。

### 日志路径

日志默认存放于 `E:\HanakoWorkSpace\skills\卦理日志\` 目录下。如果该目录不存在，会自动创建。换电脑使用时，将 `E:\HanakoWorkSpace\skills\` 整个目录复制到新电脑即可，日志路径自动跟随。

### 反馈闭环

用户事后反馈时（「上次那卦准了」「不准，实际情况是...」），将反馈追加到对应记录的 `feedback` 字段，并根据实际结果更新 `accuracy`（1-5分，5为完全准确）。

定期（每10卦或用户要求时）读取日志，分析：
- 总体准确率
- 哪类问题断得准、哪类不准

- 应期判断的偏差规律

这些分析结果可以反哺后续的解读权重。
