# 大六壬 · 四课三传排法

> 四课是天盘映射到日干支的四个信息层。三传是从四课中提取的三个关键演变节点。
> 来源：六壬大全、大六壬心镜

---

## 一、日干支寄宫

天干寄于特定地支，排四课的第一步是找到日干的寄宫：

| 天干 | 寄宫 | 天干 | 寄宫 |
|------|------|------|------|
| 甲 | 寅 | 己 | 未 |
| 乙 | 辰 | 庚 | 申 |
| 丙 | 巳 | 辛 | 戌 |
| 戊 | 巳 | 壬 | 亥 |
| 丁 | 未 | 癸 | 丑 |

**口诀**：甲寄寅乙寄辰，丙戊寄巳丁己未，庚寄申辛寄戌，壬寄亥癸寄丑。

---

## 二、四课排法

以日干支为起点，逐课取天盘之字：

1. **第一课**：日干寄宫 → 取该宫天盘上的地支 → 此为第一课上神
2. **第二课**：第一课上神 → 以其为地盘位置 → 取其天盘上的地支 → 此为第二课上神
3. **第三课**：日支（地支配对）→ 取该宫天盘上的地支 → 此为第三课上神
4. **第四课**：第三课上神 → 以其为地盘位置 → 取其天盘上的地支 → 此为第四课上神

**格式**：
```
第一课：日干[某]寄[某宫] → 上神[某]
第二课：[某] → 上神[某]
第三课：日支[某] → 上神[某]
第四课：[某] → 上神[某]
```

## 三、贼克（上下相克）

四课排出后，看各课上下神之间的五行生克关系。

- **下克上**：地盘之神克天盘之神，称为「贼」
- **上克下**：天盘之神克地盘之神，称为「克」

从四课中找贼克，按以下优先级取用：

1. 先取「贼」（下克上），有多个则用比用法
2. 无贼则取「克」（上克下），有多个则用比用法

---

## 四、九宗门三传发用

### 1. 贼克法（有贼有克）

- 取四课中唯一的贼/克为初传
- 初传到地盘上，再取该地盘的天盘字为中传
- 中传到地盘上，再取其天盘字为末传

### 2. 比用法（多个贼克）

多个贼克时，取与日干**比和**（五行相同）者为初传。其余同贼克法。

### 3. 涉害法（多个同类型比用）

上一步比用后仍有多个，取涉害深者（在地盘上所历克害多者）。

### 4. 遥克法（四课无克）

四课全无贼克时：
- 取日干遥克之神为初传（日干克某上神）
- 或取遥克日干之神（某上神克日干）

### 5. 昴星法（四课无克无遥）

完全无克无遥时，取昴星：
- 阳日：取地盘酉上之神为初传，日支上神为中传，日干上神为末传
- 阴日：取天盘酉下之神为初传，日干上神为中传，日支上神为末传

### 6. 别责法（四课不全）

四课只有三课时：
- 阳日：取日干合神之上神为初传，日干上神为中末传
- 阴日：取日支三合局前一位之上神为初传，日干上神为中末传

### 7. 八专法（干支同位）

日干支寄宫相同（四课有两课相同）：
- 阳日：以日干上神顺数三位为初传，日干上神为中末传
- 阴日：以日支上神逆数三位为初传，日干上神为中末传

### 8. 伏吟法（天盘与地盘完全相同）

天盘各宫与地盘地支相同：
- 有克：取四课中克者为初传
- 无克：阳日取日干上神为初传，刑为中传，冲为末传；阴日取日支上神

### 9. 反吟法（天盘与地盘全部相冲）

天盘各宫与地盘地支全部对冲：
- 有克：取克者为初传
- 无克：取驿马为初传

## 五、三传断法速查

三传代表事情发展的三个阶段：

| 传 | 含义 | 判断重点 |
|----|------|---------|
| 初传 | 事之始、发端 | 看五行生克谁先动 |
| 中传 | 事之中、过程 | 看有无转折 |
| 末传 | 事之终、结果 | 看最终生克关系，归何处 |

**三传吉凶趋势**：
- 初生中、中生末 → 一路顺畅
- 初克中、中克末 → 一路阻碍
- 末传生日干 → 终归有利
- 末传克日干 → 终归不利
