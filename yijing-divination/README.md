# ☯ 本地易经占卜 · Yijing Local Divination

纯本地运行的易经占卜技能。**不调用任何外部 API，不需要注册任何服务，零隐私风险。** 您的占卦问题只留在您自己的电脑上。

> 对比市面上需要 API Key、会把问题文本发送到第三方服务器的占卜技能——这个 skill 是另一个路数。

---

## 功能

- **铜钱起卦**：模拟三枚铜钱六次投掷，自动区分老阴(6)/少阳(7)/少阴(8)/老阳(9)
- **数字起卦**：随口报三个数即可起卦
- **梅花易数 8 种起卦法**：年月日时、物数、声音、字占、方位、颜色、动物、静物
- **自动识别卦象**：六十四卦本卦 + 变卦 + 互卦，含二进制编码
- **384 爻辞完整收录**：按动爻自动引用对应爻辞
- **体用生克分析**：含四时旺衰、体党用党、互变卦判断
- **八卦万物类象**：每卦 20+ 维度（天时、地理、人物、身体、婚姻、求财、疾病……）
- **纳甲八宫基础**：世应、纳地支、装六亲、装六神
- **分类占断指南**：婚姻、求财、求名、疾病、官讼、出行、失物、行人、家宅、生产
- **应期判断**：含干支换算脚本，支持旬空、月破、合冲、入墓等应期规则

## 安装

### Hanako / Claude Code 等平台

将本仓库克隆或下载到您的 skills 目录：

```bash
# Hanako 工作空间
cd 您的Hanako工作空间/.agents/skills/
git clone https://github.com/您的用户名/yijing-divination.git
```

或直接下载 zip 解压到 skills 目录。

### 依赖

- Python 3.x（仅 `scripts/ganzhi.py` 需要，用于应期判断的干支换算）
- 无需任何第三方库、无需 API Key、无需网络

## 使用

安装后直接对 AI 说：

> 「帮我算一卦，最近工作能不能有突破」

> 「用梅花易数帮我看看这次出行顺利吗」

> 「测一下我和对方这件事的发展趋势」

AI 会自动触发此技能，引导您补充必要信息后起卦解卦。
